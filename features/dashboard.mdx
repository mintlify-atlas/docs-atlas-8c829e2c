The Dashboard is the first screen you see when you open WP Manager Pro. It gives you an instant read on your site's health, resource usage, and content — all on one page without navigating elsewhere.

## How to access it

<Steps>
  <Step title="Open WP Admin">
    Log in to your WordPress admin panel.
  </Step>
  <Step title="Click WP Manager in the sidebar">
    Find **WP Manager** in the left-hand WordPress admin menu and click it. The Dashboard loads automatically as the default view.
  </Step>
  <Step title="Refresh on demand">
    Click the **Refresh** button in the top-right corner to pull the latest stats without reloading the page. The Dashboard also auto-refreshes every 30 seconds.
  </Step>
</Steps>

## Alert banner

If any of the following conditions are active, a yellow alert banner appears at the top of the Dashboard:

| Alert | Meaning | Action |
|-------|---------|--------|
| Maintenance mode is active | Your site is showing a maintenance page to all visitors | Click **Manage →** to go to the Maintenance page |
| Debug mode is enabled | `WP_DEBUG` is on — only appropriate in development | Click **Manage →** to go to Debug Tools |
| Updates available | One or more plugins, themes, or WordPress core has a pending update | Counts are broken out by type |

## Site status panel

The **Site Status** card shows the current values for key site properties:

| Field | Description |
|-------|-------------|
| WordPress | Current WP version |
| PHP | Current PHP version |
| Active Theme | Name of the currently active theme |
| Debug Mode | `Enabled` (amber warning) or `Disabled` (green check) |
| Maintenance | `Active` (amber warning) or `Off` (green check) |

## System resources panel

The **System Resources** card surfaces PHP and server limits that commonly affect site behaviour:

| Field | What it shows |
|-------|---------------|
| Memory Limit | PHP `memory_limit` (`ini_get`) |
| Upload Max | `upload_max_filesize` — maximum single file upload size |
| Max Exec Time | `max_execution_time` in seconds |
| Database Size | Total size of all tables in the current database |
| Uploads Size | Size of the `wp-content/uploads` folder |

<Tip>
If you see a low memory limit (e.g. 64M or 128M) and plugins are crashing, increase `memory_limit` in your `php.ini` or `wp-config.php` (`define('WP_MEMORY_LIMIT', '256M')`).
</Tip>

## Recent posts panel

The **Recent Posts** card lists the most recently published posts with their titles and relative timestamps. Each title is a link that opens the post on the front end in a new tab.
