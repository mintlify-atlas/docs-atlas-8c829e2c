## Tech stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19 + TypeScript + Vite |
| Styling | Tailwind CSS + shadcn/ui (Radix UI) |
| State | TanStack Query v5 |
| Editor | Monaco Editor (VS Code engine) |
| Backend | PHP 7.4+ WordPress REST API |
| Auth | WordPress nonce (`wp_rest`) |
| Namespace | `wp-manager-pro/v1` |

## Requirements

| Requirement | Minimum |
|-------------|---------|
| WordPress | 5.9 |
| PHP | 7.4 |
| MySQL / MariaDB | 5.6+ |
| Browser | Modern (ES2020+) |

<Note>
All REST endpoints are protected by the `manage_options` capability. Only WordPress Administrators can access WP Manager Pro.
</Note>
